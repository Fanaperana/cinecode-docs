## Goal

Start a new video project with a working scene you can render immediately.

## Scaffold it

```bash
cinecode new my-video
```

This creates:

```text
my-video/
├── codescene.toml      # manifest: scenes, theme, output settings
├── scenes/
│   └── intro.ccs       # a starter scene (Rhai script)
└── src/
    └── main.rs         # sample source code shown in the scene
```

## Add a scene

1. Create `scenes/outro.ccs` with a `scene(...)` call.
2. Register it in the manifest, in play order:

```toml title="codescene.toml"
scenes = ["intro", "outro"]
```

<Callout type="warn">
  Only scenes listed in `scenes = [...]` are rendered, and they play in the order
  you list them. A `.ccs` file that isn't listed is ignored.
</Callout>

## Validate before rendering

```bash
cinecode check my-video
```

`check` parses the manifest and every scene script and reports errors without
rendering — fast feedback while you write.
