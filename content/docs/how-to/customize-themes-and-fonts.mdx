## Goal

Change the overall look — palette, code colors, and typography — to match the
tone of your video.

## Set the project theme

The project theme controls the background, palette, and default code theme. Set
it in the manifest:

```toml title="codescene.toml"
theme = "dark_documentary"
```

Available project themes:

- `dark_documentary` (default) — navy background, gold accents
- `hacker_terminal` — green-on-black
- `scientific` — clean, light, neutral
- `blueprint` — blue technical drawing
- `modern_minimal` — restrained and bright
- `retro_computer` — amber CRT
- `ancient_manuscript` — parchment and ink

## Choose a code highlight theme

`s.code()` accepts an optional theme argument to override the default just for
that block. The clip below styles this real file from the example project, so
you can see exactly what each theme does to the same code:

```rust title="src/editor_theme.rs"
// blend two hex channels by a factor t
fn blend(a: u32, b: u32, t: f32) -> u32 {
    let v = a as f32 * (1.0 - t) + b as f32 * t;
    return v.round() as u32;
}

fn main() {
    let theme = "ocean";
    println!("{theme} = {}", blend(0x10, 0xF0, 0.5));
}
```

```rust
// code(source, lang, [theme], x, y)
let a = s.code(read("src/editor_theme.rs"), "rust", "base16-ocean.dark", 96, 160);
```

Built-in code themes include `base16-ocean.dark` (default),
`base16-eighties.dark`, `base16-mocha.dark`, `Solarized (dark)`,
`base16-ocean.light`, `Solarized (light)`, and `InspiredGitHub`.

List everything available on your build:

```bash

```

## Set fonts

Declare fonts in the manifest (remember: bare keys before any `[table]`):

```toml title="codescene.toml"
fonts = ["assets/fonts/Inter.ttf", "assets/fonts/FiraCode.ttf"]
```

Then select a font per node:

```rust
s.text("Chapter One", 96, 64).font("Inter").font_size(56);
s.code(read("src/main.rs"), "rust", 96, 160).font("Fira Code");
```

  `fonts` entry.
</Callout>
